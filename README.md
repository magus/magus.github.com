## setup
- `git clone https://github.com/magus/magus.github.com.git`
- `cd magus.github.com`
- `curl -sSL https://get.rvm.io | bash -s stable --ruby`
- `gem install bundler`
- `bundle install`
- Run the server with `bundle exec jekyll serve`
- Open `http://localhost:4000/` in your browser
